# Vagrant
> - 쉬운 Provisioning을 제공한다.

## Vagrant 기초 Command


- vagrant init
> Provisioning 해주는 예제 Script를 생성한다. 

- vagrant up
> vargratfile을 읽어 Provisioning을 진행한다.  


- vagrant halt
> Vagrant에서 사용하는 Host들을 종료한다.

- vagrant destory
> Vagrant의 Host들을 삭제한다. 

- vagrant ssh
> Vagrant의 Hos에 SSH로 접속한다. 


- vagrant provision
> - Vagrant의 Host의 설정 변경을 적용한다. 
> - Vgrant의 Halt와 UP을 하지 않더라도 설정 변경 사항을 바로 적용해준다.

# Vagrant를 사용한 Ansible Core 환경 구성

- Vagrantfile 파일 수정 

> 1. Vagrant에서 구분하는 Host 이름 작성
> 2. Virtual Box에서 구분하는 Host 이름 작성
> 3. Virtual Machine의 Host 이름 작성
> 4. Host PC와 Virtual Machine 간에 공유 Directory 미사용 설정
> 5. Virutal Machine에서 Internet으로 연결되는 IP 설정
> 6. Host PC의 Port를 IP 주소와 유사하게 변경

- Bootstrap.sh 생성

> 1. yum을 사용하여 EPEP 설치
> 2. yum을 사용하여 ansible 설치


## Vagrantfile 수정

>
~~~bash
# -*- mode: ruby -*-
# vi: set ft=ruby :

# All Vagrant configuration is done below. The "2" in Vagrant.configure
# configures the configuration version (we support older styles for
# backwards compatibility). Please don't change it unless you know what
# you're doing.
Vagrant.configure("2") do |config|
  config.vm.define:"ansible-server" do |cfg|
     cfg.vm.box = "centos/7"
	 cfg.vm.provider:virtualbox do |vb|
	   vb.name="Ansible-Server(github_SysNet4Admin)"
	 end
	 cfg.vm.host_name="ansible-server"
	 cfg.vm.synced_folder ".", "/vagrant", disabled: true
	 cfg.vm.network "public_network", ip: "192.168.35.10"
	 cfg.vm.network "forwarded_port", guest: 22, host: 19210, auto_correct: false, id: "ssh"
	 cfg.vm.provision "shell", path: "bootstrap.sh"
  end
  
end
~~~
- 하나의 do가 나오면 하나의 end로 마무리된다.
- **"config.vm.define"** : Vagrant에서 SSH 접속을 할 때 사용하는 Host 이름
- **"cfg.vm.box"** : 설치할 Virtual Machine의 OS
- **"cfg.vm.provider"** : Virtual Box의 일분 Config를 변경
- **"vb.name"** : Virtual Box에서 보여지는 Virtual Machine의 이름
- **"cfg.vm.host_name"** : 설치할 Virtual Machine에서 보여지는 Host 이름
- **"cfg.vm.synced_folder"** : Virtual Box가 설치되어 있는 Window와 같은 Host와 그 위에 설치되는 Virutal Machine간에 공유 Directory를 설정 (사용않함으로 설정)
- **"cfg.vm.network"** : 인터넷연결을 위한 설정 Public_network 는 Virtual Box의 Bridge Network로 설정되며 ip는 현재 내가 사용하고 있는 Internet Range의 IP 중 하나를 입력한다
- **"cfg.vm.provision"** : Shell을 호출하여 생성한 bootstrap.sh를 실행한다.


## bootstrap.sh 파일 생성

>
~~~bash
#! /usr/bin/env bash

yum install epel-release -y
yum install ansible -y
~~~
- 해당 파일을 Vagrantfile과 동일한 위치에 저장해야 한다.
