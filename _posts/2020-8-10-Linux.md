# awk

- 데이터를 조작하고 리포트를 생성하기 위해 사용하는 언어
- 리눅스에서 사용하는 awk는 GNU Version의 gawk로 심볼릭 링크되어 있다.

<br>

~~~
$ which wk   
/usr/bin/awk
 
$ ls -l /usr/bin/awk
lrwxrwxrwx 1 root root 4 Jul 24 16:01 /usr/bin/awk -> gawk
~~~

<br>

## Version
- awk : 전통적인 awk
- nwak : New awk
- gawk : GNU awk

<br>    
  
## awk 용어

- 레코드 : 한번에 처리되는 데이터의 양 (기본값 : 라인)
- 필드 : 필드 부로 분리된 레코드의 내의 데이터
