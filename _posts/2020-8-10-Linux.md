# awk

- 데이터를 조작하고 리포트를 생성시키는데 사용하는 프로그램 언어
- Line, Field 단위로 처리 가능
- 패턴 매칭 기능 사용
- grep 이나 sed 와 동작 방법이 유사

-   -   -

## awk Version

- awk : 전통적인 awk
- nwak : New awk
- gawk : GNU awk
